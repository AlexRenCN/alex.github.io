JPS

jps列出目标系统上已经检测的JAVA虚拟机，通常用于查询当前服务器中正在运行的java虚拟机进程信息。

jps的详细参数如下

  • -q
    只列出进程号而不列出其他信息

  • -m
    列出进程号、名称以及传递给main方法的参数

  • -l
    列出进程号、名称以详细路径

  • -v
    列出进程号、名称以及JVM的详细参数

  • -V
    列出进程号、名称

  • -J option
    将option传递给JVM，比如-J-Xms48m将启动内存设置为48 MB，不过该功能支持并不完善
